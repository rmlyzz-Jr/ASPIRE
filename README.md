# 🚀 ASPIRE - Sistem Informasi Pengelola Aduan DPU

**ASPIRE** adalah aplikasi web PWA untuk mengelola pengaduan infrastruktur di Kota Semarang.

## ✨ Fitur

- 📝 Buat Laporan dengan keyword detection & upload gambar
- 📋 Daftar Laporan dengan filter & export
- 📊 Dashboard real-time & peringkat bidang
- 📈 Analytics & Prediction dengan AI-powered
- 🗺️ Peta Sebaran Aduan per Kecamatan
- 🚨 Early Warning System & Hotspot Detector
- 👤 Admin Panel manajemen user
- 📱 PWA Support (install sebagai aplikasi)

## 🛠️ Teknologi

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Google Apps Script
- **Database:** Google Sheets
- **Library:** Chart.js, Font Awesome, Google Maps API
- **PWA:** Service Worker, Manifest
